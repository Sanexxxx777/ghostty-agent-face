# Security Policy

## Scope

This project is a local shell script + Ghostty GLSL shader. It writes OSC 11
background-color escape sequences to your own terminal tty and reads them
back in the shader — there is no network activity, no daemon, and no data
collected or transmitted anywhere.

## Reporting a vulnerability

If you find a security issue, please open a GitHub issue or contact
[@Aleksandr_NFA](https://github.com/Sanexxxx777).

## Supported versions

Latest `main` only.
